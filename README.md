## Info
Compares two files and outputs a difference.\
Supported formats: json, yaml

## Requirements

Node.js : `^22.13.0`

## Installation

1. Clone the repository.
```shell
git clone https://github.com/Droletor/frontend-project-46.git
```

2. Install the dependencies.
```shell
make install
```

## Usage

Get Information
```shell
gendiff --help
```
